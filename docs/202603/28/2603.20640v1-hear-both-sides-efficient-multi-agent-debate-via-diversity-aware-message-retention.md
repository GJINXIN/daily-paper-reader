## 摘要
多智能体辩论（Multi-Agent Debate）已成为一种通过智能体间的迭代通信来提高大语言模型推理质量的有前景的框架。然而，在每一轮中广播所有智能体的消息会引入噪声和冗余，这可能会降低辩论质量并浪费计算资源。目前的方法依赖于不确定性估计，在广播前过滤低置信度的响应，但由于置信度分数校准不准以及对阈值选择的敏感性，这种方法并不可靠。为解决这一问题，我们提出了多样性感知保留（Diversity-Aware Retention, DAR），这是一种轻量级的辩论框架，它在每一轮辩论中，在广播前选择一组彼此之间以及与多数投票结果分歧最大的智能体响应子集。通过一种显式的基于索引的保留机制，DAR 在不进行修改的情况下保留原始消息，确保保留的分歧保持真实。在多种推理和问答基准测试上的实验表明，我们的选择性消息传播一致地提高了辩论性能，特别是在智能体数量增加、噪声累积最严重的情况下。我们的结果强调，在多智能体推理系统中，智能体“听到”什么与智能体“说”什么同样重要。

## Abstract
Multi-Agent Debate has emerged as a promising framework for improving the reasoning quality of large language models through iterative inter-agent communication. However, broadcasting all agent messages at every round introduces noise and redundancy that can degrade debate quality and waste computational resources. Current approaches rely on uncertainty estimation to filter low-confidence responses before broadcasting, but this approach is unreliable due to miscalibrated confidence scores and sensitivity to threshold selection. To address this, we propose Diversity-Aware Retention (DAR), a lightweight debate framework that, at each debate round, selects the subset of agent responses that maximally disagree with each other and with the majority vote before broadcasting. Through an explicit index-based retention mechanism, DAR preserves the original messages without modification, ensuring that retained disagreements remain authentic. Experiments on diverse reasoning and question answering benchmarks demonstrate that our selective message propagation consistently improves debate performance, particularly as the number of agents scales, where noise accumulation is most severe. Our results highlight that what agents hear is as important as what agents say in multi-agent reasoning systems.

#### 2. 方法论：核心思想与技术细节
论文提出了 **多样性感知保留（Diversity-Aware Retention, DAR）** 框架，其核心思想是：**在每一轮辩论中，仅选择并广播那些彼此之间以及与当前多数意见分歧最大的响应子集。**

**关键技术组件：**
*   **不确定性评分（ANLL）**：计算答案 Token 的平均负对数似然（Average Negative Log-Likelihood），并将其作为辅助信息附加在 Prompt 中，帮助智能体评估他人响应的可靠性。
*   **多数投票锚点（Majority Vote）**：将前一轮的多数投票结果提取并置于 Prompt 头部，作为显式的共识参考点。
*   **多样性过滤模块（Filter Agent）**：
    *   引入一个轻量级的过滤智能体（通常与推理智能体同质）。
    *   该智能体接收所有响应，目标是输出一组**智能体索引（IDs）**，这些索引对应的响应应具有最大的互异性和对多数派的挑战性。
    *   **索引机制**：仅输出 ID 而非重写内容，确保原始辩论信息的真实性和完整性，避免小模型在编辑过程中引入额外错误。
*   **回退机制**：若所有智能体意见高度一致（无显著分歧），则回退到标准 MAD 模式。

#### 6. 主要结论与发现
*   **DAR 显著优于基准**：在所有模型和数据集上，DAR 一致性地提升了辩论性能。
*   **规模效应**：随着智能体数量 $N$ 的增加，标准 MAD 的性能往往因噪声积累而饱和甚至下降，而 DAR 在 $N=8$ 时表现出更强的鲁棒性和持续的性能增益。

#### 7. 优点（亮点）
*   **轻量且即插即用**：无需微调模型，通过简单的 Prompt 工程和索引过滤即可实现。
*   **索引保留机制**：巧妙地解决了 LLM（尤其是小模型）在总结或编辑他人观点时可能产生的幻觉问题，确保了信息的“原汁原味”。
*   **解决共识陷阱**：通过强制保留分歧，有效防止了多智能体系统过早陷入错误的共识。

#### 8. 不足与局限
*   **过滤智能体的依赖**：DAR 的效果高度依赖于过滤模块识别“分歧”的能力。如果底座模型极弱，可能无法有效筛选出高质量的异见。
*   **推理延迟**：虽然减少了广播的消息量（节省了后续轮次的 Token），但在每轮开始前增加了一个过滤步骤，会带来微小的额外推理延迟。
*   **长文本挑战**：尽管减少了冗余，但在极端复杂的问题下，保留多个长篇大论的分歧意见仍可能触及小模型的上下文处理极限。
