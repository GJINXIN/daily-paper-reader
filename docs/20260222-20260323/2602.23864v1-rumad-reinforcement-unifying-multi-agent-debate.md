## 摘要
多智能体辩论（MAD）系统利用集体智慧来增强推理能力，但现有方法难以同时优化准确性、共识形成和计算效率。静态拓扑方法缺乏对任务复杂度变化的适应性，而基于外部大语言模型（LLM）的协调则存在引入特权知识的风险，从而损害辩论的中立性。本研究提出了 RUMAD（Reinforcement-Unifying Multi-Agent Debate），这是一个将 MAD 中的动态通信拓扑控制建模为强化学习（RL）问题的新型框架。RUMAD 采用了一种与内容无关的观测方案，捕捉高层辩论动态，避免访问智能体的原始推理内容。RUMAD 使用多目标奖励来建模解决方案的质量、凝聚力和效率。一个经过 PPO 训练的控制器动态调整通信图中的边权重，而双阈值机制实现了对智能体激活和信息可见性的细粒度控制。在 MMLU、GSM8K 和 GPQA 基准测试上的实验评估表明，RUMAD 实现了显著的效率提升，将 Token 成本降低了 80% 以上，同时与单个 LLM 模型和多个 MAD 基准相比，仍提高了推理准确性。值得注意的是，仅在 MMLU 上训练的 RUMAD 在域外（OOD）任务上表现出鲁棒的零样本泛化能力，这表明所学习的通信策略捕捉到了有效多智能体协调中与任务无关的原则。这些结果确立了 RUMAD 是一种在实际资源约束下部署多智能体推理应用的高效且鲁棒的方法。

## Abstract
Multi-agent debate (MAD) systems leverage collective intelligence to enhance reasoning capabilities, yet existing approaches struggle to simultaneously optimize accuracy, consensus formation, and computational efficiency. Static topology methods lack adaptability to task complexity variations, while external LLM-based coordination risks introducing privileged knowledge that compromises debate neutrality. This work presents RUMAD (Reinforcement-Unifying Multi-Agent Debate), a novel framework that formulates dynamic communication topology control in MAD as a reinforcement learning (RL) problem.   RUMAD employs a content-agnostic observation scheme that captures high-level debate dynamics avoiding access to raw agent reasoning content. RUMAD uses a multi-objective reward to model solution quality, cohesion and efficiency. A PPO-trained controller dynamically adjusts edge weights in the communication graph, while a dual-threshold mechanism enables fine-grained control over both agent activation and information visibility.   Experimental evaluation across MMLU, GSM8K, and GPQA benchmarks demonstrates that RUMAD achieves substantial efficiency gains, reducing token costs by over 80\%, while still improving reasoning accuracy compared to single LLM model and multiple MAD baselines. Notably, RUMAD trained exclusively on MMLU exhibits robust zero-shot generalization to out-of-domain (OOD) tasks, indicating that the learned communication strategies capture task-independent principles of effective multi-agent coordination. These results establish RUMAD as a efficient and robust approach for deploying multi-agent reasoning application with practical resource constraints.

### 1. 核心问题与整体含义（研究动机和背景）
多智能体辩论（Multi-Agent Debate, MAD）通过多个大语言模型（LLM）的交互来提升推理能力，但面临三大挑战：
*   **缺乏适应性**：现有方法多采用静态通信拓扑（如环形、星形），无法根据任务难度动态调整资源分配。
*   **中立性风险**：引入外部 LLM 作为“裁判”或“协调者”会引入外部偏见，破坏辩论的自主涌现性。
*   **计算效率低下**：缺乏对 Token 成本的显式建模，导致在简单任务上浪费资源，或在复杂任务上通信不足。
**RUMAD 的核心目标**是利用强化学习（RL）动态控制辩论拓扑，在保证中立性的前提下，实现准确性、共识形成与计算效率的最优平衡。

### 2. 方法论：核心思想与关键技术
RUMAD 将动态拓扑控制建模为一个马尔可夫决策过程（MDP），并使用 PPO 算法进行训练。
*   **内容无关观测（Content-Agnostic Observation）**：控制器不读取智能体的原始文本，仅观测由语义相似度（余弦相似度）和答案一致性构成的 $N \times N$ 矩阵。这种设计确保了控制器的中立性，使其学习的是“协调策略”而非“领域知识”。
*   **动作空间与分布建模**：控制器输出智能体间连接权重的概率分布参数（均值 $\mu$ 和标准差 $\sigma$），通过高斯分布采样并经过 Sigmoid 激活得到边权重 $w_{ij}$。
*   **双阈值通信机制**：
    *   **激活阈值**：若某智能体受到的外部影响权重低于阈值，则该轮被“静默”（复用上一轮输出），从而节省 Token。
    *   **可见性阈值**：根据权重大小将邻居信息标记为“关键”、“参考”或“背景”，实现精细化的 Prompt 构建。
*   **多目标奖励函数**：整合了准确性（Accuracy）、共识度（Consensus）、推理进步（Progress/Improvement）、Token 效率（Efficiency）和拓扑稀疏性（Sparsity）五个维度。

*   **极高的效率提升**：相比全连接 MAD，RUMAD 在 MMLU 和 GSM8K 上**降低了超过 80% 的 Token 成本**。

*   **预算可控性**：通过调整训练时的预算参数 $B$，可以灵活引导模型在“高性能”和“极低成本”之间切换。

### 7. 优点（亮点）
*   **创新的内容无关设计**：通过只看“元数据”（相似度）而不看“内容”来控制辩论，有效避免了协调者的偏见。
*   **端到端的效率优化**：将 Token 成本直接纳入 RL 奖励函数，是目前 MAD 研究中少有的对计算资源进行显式建模的工作。
*   **零样本迁移能力**：证明了多智能体协作策略具有跨领域的通用性，具有很强的工程实践价值。

### 8. 不足与局限
*   **中心化瓶颈**：虽然 PPO 控制器很轻量，但在智能体规模扩展到数百个时，中心化处理 $N \times N$ 矩阵可能会产生计算压力。
